# Lewis Muithya's Travel Log

A sleek, interactive travel log web app built with **HTML, CSS, and JavaScript**, styled in a **Jet Black + Tang Orange hypercar-inspired theme**. Users can view visited places, see details with smooth animations, and add new destinations through a responsive form.

---

## ‚ú® Features

- **Visited Places List**  
  Displays destinations with hover glow effects and smooth transitions.

- **Place Details Viewer**  
  Clicking a place fades in its details with a Tang Orange glow accent.

- **Add New Place Form**  
  Responsive two-column form for adding new destinations, landmarks, time of year, and notes.

- **Animations & Effects**  
  - Page sections fade/slide in on load.  
  - Header racing stripe slides across and pulses like neon.  
  - Buttons and list items glow in Tang Orange on hover.  
  - Place details fade in with a glow when selected.

- **Responsive Design**  
  Works seamlessly on desktop and mobile, with form inputs stacking neatly on smaller screens.
